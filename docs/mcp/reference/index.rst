Tool Reference
==============

**74 tools** across 10 plugin modules, each independently installable. Install only the
modules that match the Odoo apps deployed in your instance.

.. list-table::
   :header-rows: 1
   :widths: 25 20 10 45

   * - Module
     - Odoo App
     - Tools
     - Key Capabilities
   * - ``lu_mcp_knowledge``
     - Knowledge
     - 9
     - Browse workspaces, read / search / create / update / move / delete articles,
       partial edits via ``content_updates``
   * - ``lu_mcp_project``
     - Project
     - 7
     - List projects and stages, search / read / create / update tasks,
       link Knowledge articles
   * - ``lu_mcp_helpdesk``
     - Helpdesk
     - 12
     - Full ticket lifecycle — create, reply, escalate, close, SLA status,
       customer history, link Knowledge articles
   * - ``lu_mcp_crm``
     - CRM
     - 8
     - Pipeline summary, leads / opportunities — create, update, convert,
       stage list, internal notes
   * - ``lu_mcp_contact``
     - Contacts
     - 6
     - Search, read, create, update contacts and companies, list sub-contacts,
       log internal notes
   * - ``lu_mcp_calendar``
     - Calendar
     - 5
     - List, read, create, update, delete events; recurring event guard
   * - ``lu_mcp_sale``
     - Sales
     - 6
     - Search, read, create, update, confirm sales quotations and orders,
       log notes
   * - ``lu_mcp_purchase``
     - Purchase
     - 6
     - Search, read, create, update, confirm purchase orders / RFQs,
       log notes
   * - ``lu_mcp_timesheet``
     - Timesheets
     - 6
     - Log / read / update / delete timesheet entries, project summary
   * - ``lu_mcp_survey``
     - Surveys
     - 6
     - List / read / create surveys, add questions (9 types), view responses,
       generate from Knowledge articles
