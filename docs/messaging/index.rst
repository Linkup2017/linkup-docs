Messaging
=========

KakaoTalk, Alimtalk, and SMS integrations for Odoo.

.. list-table::
   :header-rows: 1
   :widths: 35 65

   * - Repository
     - Description
   * - ``lu_sms``
     - SMS gateway connector (multi-provider)
   * - ``lu_kakao``
     - KakaoTalk messaging integration
   * - ``lu_alimtalk``
     - Alimtalk (알림톡) template-based notifications

.. note::

   Detailed documentation for Messaging modules is planned for Phase 2
   (W10+).
