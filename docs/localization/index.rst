Localization (KR)
=================

Korean localization modules for Odoo — tax invoices, payroll, social insurance,
and regulatory compliance.

.. list-table::
   :header-rows: 1
   :widths: 35 65

   * - Repository
     - Description
   * - ``l10n_kr``
     - Korean chart of accounts and tax configuration
   * - ``lu_l10n_kr``
     - Tax invoice (세금계산서), e-filing, social insurance (4대보험)
   * - ``lu_l10n_kr_hr_payroll``
     - Korean payroll — income tax, deductions, pay slip templates

.. note::

   Detailed documentation for Localization modules is planned for Phase 2
   (W10+). This page will be expanded with installation guides, configuration
   steps, and usage examples.
